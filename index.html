About

My Vision

Hi! I’m Benjamin, and I created SarapMatch because I know the feeling of standing in the kitchen, staring at random ingredients, and thinking, “Ano lulutuin ko?” I wanted to make something fun, useful, and easy to use, a site that helps you turn what you already have into something delicious.

How It Works

Add Ingredients

List down what’s in your kitchen, no matter how simple.

Find Recipes

Using AI from DeepSeek and images from Unsplash, SarapMatch finds the best recipes for you.

Cook & Enjoy

Follow easy steps and enjoy your homemade meal.

What I Used to Build It

  • Bootstrap for a responsive, clean layout
  • Custom CSS for styling tweaks
  • HTML & JavaScript for structure and interactivity
  • DeepSeek API for recipe matching
  • Unsplash API for high-quality food images
